# Launch Commands

- `bootstrap`: Create v3 root, readings, manifests, claim lock, trust artifacts.
- `inspect-smoke`: Run one-sample mock Inspect eval for each mapped v3 phase.
- `audit-completion`: Recompute completion contract from manifests, ledgers, and Inspect logs.
- `run-primary-dev`: Run Inspect-native dev path for the primary denominator.
- `freeze-primary`: Freeze primary manifests, claim predicate, and task metadata.
- `run-primary-confirmatory`: Run Inspect-native confirmatory backbone path.
- `run-transfer`: Run bounded SCBench transfer path or preserve downgrade.
- `run-external`: Run tertiary external path; does not satisfy primary quotas.
- `export-conference-package`: Analyze and export final tables/reports from canonical evidence.
